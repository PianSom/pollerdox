If you like a easy-to-digest step-by-step guide, written by a Unifi Poller user for implementation on a Raspberry Pi using Docker then have a look at this [link] (https://nerdygeek.uk/2020/06/18/unifi-poller-an-easy-step-by-step-guide/)
